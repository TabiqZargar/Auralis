# styles/

Global styles and CSS configuration.

## Files

- `globals.css` — Tailwind import, CSS custom properties, base styles, utility classes

## Rules

- Use Tailwind utility classes in components whenever possible.
- CSS custom properties for theme values.
- Keep global styles minimal — prefer Tailwind.
- No scoped stylesheets — use Tailwind or CSS-in-JS (Framer Motion variants).
